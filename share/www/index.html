Source

Axios is open source, specifically aimed for teaching the Ada 2012 programming language for beginners. Please refer to the document provided with the source code.

Axios is available under a GPL v3.0 License, and the code, as well as the documentation may be accessed in the following uri:

http://github.com/psyomn/ash.git

Contributions

Contributions are always welcome to this project. However, one has to remember that this project is an educational case study that aims to demonstrate many aspects of the Ada programming language, without other noise. For example improving the software architecture of this project in order to make it a more 'useful' server is not within this scope of this case study...

However, if you have material that you wish to add to this project, then you are more than welcome to, provided that you add explanations and learning material to the given latex document (which is in the same repository).
